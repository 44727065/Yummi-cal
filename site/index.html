Welcome to Yummical datepick theme.

Let's be honest, datepick™ is, hands down, one of the best datepicker out there.
But being honest, his themes are a bit, herr… outdated… So well, here's a new one.

Here you will find:

  • A well commented CSS file, ready to be tweaked, think about it like a boilerplate.
  • One javascript snippet to adjust font size on window resize events
  • An example page
  • Nothing else

Usage is very simple:

  1. Download datepick package from developer website -> datepick
  2. Find the "jquery.datepick.css" file and subsitute it with this one.
  3. Add those lines when you call for .datepick(); 
    						
$('#your_container_here').datepick({
  // your options here
  onShow: function() { 
    // this removes an unnecessary style from .datepick DOM node
    $('.datepick').removeAttr('style'); 
  } 
});
  4. No other steps are required (unless you want to hack it a bit).
