Welcome to Yummical datepick theme.

So, datepick™ is, hands down, one of the best datepicker for jQuery out there.
But being honest, his themes are a bit, herr… outdated… So well, here's a new one.

What will I find?

  • A well commented LESS file, ready to be styled.
  • A javascript snippet to adjust font size on window resize events
  • An example page
  • Nothing else

What sould I do?

  1. Download datepick package from the developer website -> datepick
  2. Find the "jquery.datepick.css" file and change it with this one.
  3. Add those lines when you call for .datepick(); 
    						
$('#your_container_here').datepick({
  // your options here
  onShow: function() { 
    // this removes an unnecessary style from .datepick DOM node
    $('.datepick').removeAttr('style'); 
  } 
});
  4. You are done.
